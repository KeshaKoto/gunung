JENIS JENIS GUNUNG

Berdasarkan Aktivitasnya

GUNUNG API ISTIRAHAT

Deskripsi: Gunung api istirahat yaitu aktivitas gunung api yang sewaktu-waktu dapat Meletus kemudian beristirahat lagi. Atau tidak mengeluarkan erupsi dalam 10.000 tahun terakhir namun ada potensi untuk erupsi
Contoh:
Gunung Ciremai, Gunung Kelud

GUNUNG API MATI

Deskripsi:
Gunung api mati adalah gunung yang tidak mempunyai aktivitas erupsi atau tidak ada catatan letusan ribuan tahun terakhir
Contoh:
Gunung Argapuro, Gunung Arjuna

GUNUNG API AKTIF

Deskripsi: Gunung api aktif adalah gunung yang masih berkerja aktif, ditandai dengan sering mengeluarkan asap dan di daerah sekitar sering terjadi gempa dan kadang letusan.
Contoh:
Gunung Sinabung, Gunung Egon
